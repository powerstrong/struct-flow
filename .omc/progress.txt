## 학습/패턴

- **Node 22+ node:sqlite는 Vite ESM 리졸버에서 깨진다.** Test 하니스에서 createRequire로 우회.
- **z.ZodType<I>는 .default() 있는 zod object를 받지 못한다** (Input/Output 타입 불일치). registry의 Calculator 인터페이스에서 `z.ZodType<I, ZodTypeDef, unknown>`로 변경.
- **better-sqlite3는 Windows에서 Visual Studio C++ build tools 필요.** Node 22+의 node:sqlite로 대체 가능.
- **api tsconfig rootDir 설정은 monorepo path alias와 충돌.** rootDir 제거하면 path alias가 매끄럽게 동작.
- **wrangler/pnpm 미설치 환경**: npx wrangler로 호출 가능. npm workspaces로 충분.
